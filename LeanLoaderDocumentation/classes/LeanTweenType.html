LeanTweenType Class

Defined in: LeanTween.js:38

Pass this to the "ease" parameter in the optional hashtable, to get a different easing behavior

Example Javascript:
LeanTween.rotateX(gameObject, 270.0f, 1.5f, {"ease":LeanTweenType.easeInBack});

Example C#:
Hashtable optional = new Hashtable();
optional.Add("ease":LeanTweenType.easeInBack);
LeanTween.rotateX(gameObject, 270.0f, 1.5f, optional);
